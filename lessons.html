Git Lessons

Complete lessons in order. Intermediate and advanced unlock as you progress.

1. What is Git?

Git tracks changes in files over time and lets you collaborate safely with others.

2. Start a Repository

git init

This creates a new Git repository in your current project folder.

3. Check Status

git status

See untracked files, modified files, staged files, and the current branch.

4. Stage Files

git add .

Stage all current changes for the next commit.

Intermediate Lessons Locked

Complete at least 4 lessons to unlock this level.

5. Commit Changes

git commit -m "Initial commit"

Create a saved snapshot of your staged work.

6. View History

git log --oneline

See a short readable commit history.

7. Create and Switch Branches

git switch -c feature-navbar

Create a feature branch and move to it in one step.

8. Merge a Branch

git merge feature-navbar

Merge changes from a feature branch into your current branch.

Advanced Lessons Locked

Complete at least 8 lessons to unlock this level.

9. GitHub Pages Workflow

git add .
git commit -m "Update site"
git push origin main

Typical workflow for updating a GitHub Pages website.

10. Git in VS Code

Use Source Control in VS Code to stage files, write commit messages, and push changes.

11. Stash and Rebase

git stash
git rebase main

Temporarily store work and replay commits on top of the latest main.

12. Revert and Reset

git revert <commit-id>
git reset --hard HEAD~1

Learn the safe undo method and the dangerous history rewrite method.
